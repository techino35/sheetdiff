## Full Description

**SheetDiff** makes spreadsheet auditing effortless. Whether you're tracking data changes between versions, reviewing client deliverables, or validating ETL results, SheetDiff gives you a clear, color-coded view of exactly what changed — and where.

### How It Works

1. Open **SheetDiff** from the custom menu in any Google Sheet.
2. Select a **source sheet** (baseline) and a **destination sheet** (updated version).
3. Optionally specify a **key column** (e.g., "ID") for accurate row matching across shuffled data.
4. Click **Compare** — SheetDiff highlights every difference in seconds.

### Features

**Visual Diff**
- **Added rows** highlighted in green
- **Deleted rows** captured in the report (shown in red)
- **Modified cells** highlighted in yellow, with the original value saved as a cell comment

**Diff Report Sheet**
Automatically generates `_SheetDiff_Report` with:
- Cell reference (e.g., C12)
- Change type (Added / Deleted / Modified)
- Old value and new value side-by-side

**Snapshot Mode**
Save any sheet as a hidden snapshot with one click. Compare the live sheet against the snapshot later — perfect for periodic audits.

**Smart Row Matching**
LCS (Longest Common Subsequence) algorithm matches rows by key column, so insertions and deletions don't throw off the rest of the comparison.
